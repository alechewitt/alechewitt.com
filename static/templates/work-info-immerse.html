IMMERSE LEARNING PLATFORM

x

The Immerse platform is a 3D game that we sell to businesses to do their training and meetings. The game is programmed in Unity and wrapped in an AnguarJS application which loads the game as well as providing additional functionality for sharing documents and partaking in group exercises.

My responsibilities have included:

  • Leading the development of the websocket multiplayer server in Golang.
  • Developing the AngularJS application in ES6.
  • Writing the backend in Node.js.
  • Introduced the WebRTC voice solution that is used to get real time communication.
