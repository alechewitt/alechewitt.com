COLORCRUMBLE

x

Color crumble is a game to make learning addictive. It is targeted at young children to help them learn addition. My thought process revolves around basing every decision on what will make the game more fun, addictive and engaging rather than what will make the child learn more. The idea being that children will play the game for longer, have more fun and by proxy learn more.

In its current form the animations are laggy and there are quite a few bugs however I am currently working on re writing it with WebGL.

Current techstack: JavaScript and CSS animations.

Soon to be released stack: WebGL.
